# 🏭 IEM Python 視覺化教學地圖

> **工業工程與管理 × Python 實務應用**
> 以真實製造情境為主軸，從資料處理、科學計算、機器學習到網路爬蟲，循序建立工程師的資料分析能力。

---

## 📋 課程總覽

本課程共分為 **4 大模組、8 個單元**，每個單元均包含：

- 🔧 核心套件語法與觀念
- 🏭 工業工程情境實作範例
- 🖥️ 可互動的 Streamlit App
- 📝 課後練習與思考題

---

## 🗺️ 課程架構

### 模組一｜資料與影像的基礎打底

奠定 Python 資料處理與影像操作基礎，銜接製造現場常見的視覺化需求。

| 單元 | 主題 | 核心套件 | 情境應用 |
|------|------|----------|----------|
| 1-1 | [影像處理基礎](Module1_1_Pillow.md) | `Pillow` | AOI 瑕疵影像前處理器 |
| 1-2 | [QR Code 生成](Module1_2_QRCode.md) | `qrcode` | 智慧倉儲標籤產生器 |
| 1-3 | [陣列運算與品管圖](Module1_3_NumPy.md) | `NumPy` | SPC 品質管制圖（X-bar & R Chart）|

---

### 模組二｜科學計算與資料分析

運用數學最佳化與結構化資料分析，支援工廠的決策與績效管理。

| 單元 | 主題 | 核心套件 | 情境應用 |
|------|------|----------|----------|
| 2-1 | [線性規劃最佳化](Module2_1_SciPy.md) | `SciPy` | 產能最佳化決策面板 |
| 2-2 | [資料清理與分析](Module2_2_Pandas.md) | `Pandas` | OEE 設備綜合效率儀表板 |

---

### 模組三｜機器學習與 AI 啟蒙

以非監督式學習切入，讓工程師理解 AI 如何從資料中發現隱藏規律。

| 單元 | 主題 | 核心套件 | 情境應用 |
|------|------|----------|----------|
| 3-1 | [K-Means 分群](Module3_1_ScikitLearn.md) | `scikit-learn` | 供應商績效自動分群 |

---

### 模組四｜資料搜集與網路爬蟲

自動化擷取外部市場資訊，串接分析流程，強化採購與供應鏈決策。

| 單元 | 主題 | 核心套件 | 情境應用 |
|------|------|----------|----------|
| 4-1 | [基礎爬蟲](Module4_1_WebScraping.md) | `requests` + `BeautifulSoup` | 原物料價格即時看板 |
| 4-2 | [Scrapy 爬蟲框架](Module4_2_Scrapy.md) | `Scrapy` | 競品價格雷達圖分析 |

---

## 🚀 快速開始

### 環境安裝

```bash
pip install pillow qrcode numpy scipy pandas scikit-learn requests beautifulsoup4 scrapy streamlit 
```

### 執行 Streamlit App

每個單元的互動 App 均可獨立執行：

```bash
streamlit run app.py
```

## 🏭 情境貫穿說明

本課程以「智慧製造工廠」為背景，各單元場景彼此呼應：

- **現場品管** → 影像 AOI 前處理、SPC 管制圖
- **倉儲物流** → QR Code 標籤自動生成
- **生產決策** → 線性規劃產能最佳化
- **績效管理** → OEE 儀表板即時監控
- **供應鏈** → 供應商分群、競品價格爬蟲
